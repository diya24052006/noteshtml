*USAGE OF STYLE TAG2

SUB TAG IS USED TO WRITE TEXT BELOVE THE NORMAL LINE

SUP TAG IS USED TO WRITE TEXT ABOVE THE NORMAL LINE

Q TAG IS USED FOR MARKING QUOTATION(" ") IN THE PARAGRAPH

ADDRESS TAG IS USED FOR WRITING ADDRESS/CONTACT DETAILS OF THE AUTHOR

EXAMPLE:

303/C PRIDE,
VASTRAL,
3800601,
INDIA
